WHAT IT SOLVES

AI tools are scattered — keys in a dozen places, context lost between sessions, the same prompts and setups rebuilt for every project. Jojo OS puts models, memory, skills, agents, MCP tools, files and voice in one local hub, with a security model that keeps secrets and private notes out of anything you share.

FEATURES

Model HubMany providers & models, favorites, per-purpose defaults, real tests.
Jojo VaultLocal Markdown memory with pins, tags, backlinks, auto-recall.
Skills & AgentsReusable SKILL.md packs and scoped, safe agent presets.
LoopsManual / scheduled / event automations — disabled by default.
MCPPaste a link to connect servers; deny-by-default tools, per-call approval.
VoicePush-to-talk input + system speech output, fully optional.

PRIVACY & SECURITY

Jojo OS runs entirely on your computer. There is no Jojo OS backend, no login, and no telemetry — the developer cannot access your API keys, prompts, files, or memories. Keys are encrypted with your OS keychain and only sent to the AI provider you pick. MCP tools and agents are deny-by-default and dangerous actions ask for approval.

ABOUT JOJO OS

Jojo OS is an open-source personal AI command center for managing models, memory, skills, agents, MCP tools, files, voice, and local workflows from one desktop app.

Jojo OS is named after Jojo, my dog. He was my best friend, and my whole family misses him a lot. I wanted to name this project after him because the app is meant to feel loyal, helpful, and always there when you need it.
